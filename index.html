About Me

Master’s Thesis

Current research direction at the Center for Machine Vision and Signal Analysis (CMVS).

Development of a Multimodal System for Non-Contact Physiological Monitoring

My thesis investigates whether thermoregulation-driven temperature dynamics in facial and palm regions, captured through thermal imaging, can serve as reliable non-contact indicators of autonomic activity, particularly respiration-related and electrodermal responses.

The system integrates thermal imaging, RGB-D sensing, and wearable biosensors to study the relationship between skin temperature dynamics and reference physiological signals. The broader goal is to advance contactless physiological monitoring for applications in healthcare, affective computing, and intelligent human-centered systems.

Affiliation: Center for Machine Vision and Signal Analysis (CMVS), University of Oulu

Publications

Selected research output and scholarly work.

A Multichannel Localization Method for Camouflaged Object Detection

Rahman, M. R. and co-authors. Presented at the IEEE International Conference on Image Processing (ICIP 2023).

This work proposes a lightweight multichannel localization framework for camouflaged object detection using Fourier-based analysis, entropy cues, class activation maps, and global average pooling to highlight discriminative regions.

Research Projects

Projects aligned with my research direction in thermal vision, multimodal sensing, and scientific software.

ThermApp Thermal Imaging Application

A Python-based software system for the ThermApp infrared camera, including low-level USB communication, real-time frame acquisition, sensor calibration, and temperature extraction using Planck-law-based conversion.

The system was extended with thermal face detection to support automatic region-of-interest processing in thermal video streams and research-driven analysis workflows.

GitHub Repository

Multimodal Non-Contact Physiological Monitoring

Ongoing research on whether facial and palm thermal dynamics can be used as contactless markers of physiological activity when analyzed together with additional sensing modalities.

The work combines thermal imaging, RGB-D sensing, and wearable references to investigate temporal patterns related to autonomic responses and respiratory behavior.

Multichannel Localization for Camouflaged Object Detection

A research project proposing a computationally lightweight localization method for camouflaged objects using Fourier phase-amplitude analysis, local entropy, and class activation maps.
